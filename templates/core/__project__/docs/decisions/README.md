# Architecture Decision Records

| ADR | Title | Status |
|-----|-------|--------|
| ADR-001 | _(your first decision)_ | _Draft_ |

ADR format (one file per significant decision):

```markdown
# ADR-NNN — <Decision Title>

**Context:** <What situation forced a decision>  
**Decision:** <What we chose>  
**Alternatives:** <What we considered and rejected, and why>  
**Consequences:** <What this means going forward — costs and benefits>
```

Filenames: `ADR-NNN-kebab-subject.md` (e.g. `ADR-001-static-content.md`). The `ADR-` prefix is intentional and exempt from any project-wide kebab-case-only filename rule.
